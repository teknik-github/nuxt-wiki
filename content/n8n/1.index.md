# Pengenalan n8n
n8n adalah alat otomasi alur kerja sumber terbuka yang memungkinkan Anda menghubungkan berbagai aplikasi dan layanan untuk mengotomatiskan tugas-tugas rutin. Dengan n8n, Anda dapat membuat alur kerja yang kompleks tanpa perlu menulis kode, menggunakan antarmuka visual yang intuitif.

## Fitur Utama n8n
- **Sumber Terbuka**: n8n adalah proyek sumber terbuka, yang memungkinkan pengguna untuk mengakses dan memodifikasi kode sumber sesuai kebutuhan mereka.
- **Antarmuka Visual**: n8n menyediakan antarmuka drag-and-drop yang memudahkan pembuatan alur kerja tanpa perlu menulis kode.
- **Integrasi Luas**: n8n mendukung integrasi dengan ratusan aplikasi dan layanan populer, termasuk Google Sheets, Slack, GitHub, dan banyak lagi.
- **Eksekusi Kondisional**: n8n memungkinkan Anda menambahkan logika kondisional ke alur kerja Anda, sehingga Anda dapat mengontrol aliran data berdasarkan kondisi tertentu.
- **Eksekusi Paralel**: n8n mendukung eksekusi paralel, memungkinkan Anda menjalankan beberapa tugas secara bersamaan untuk meningkatkan efisiensi.
- **Ekstensi dan Kustomisasi**: n8n memungkinkan Anda membuat node kustom dan ekstensi untuk menyesuaikan fungsionalitas sesuai kebutuhan spesifik Anda.

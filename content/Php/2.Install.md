Untuk menginstal PHP, ikuti langkah-langkah berikut sesuai dengan sistem operasi yang Anda gunakan:
## **1. Installasi PHP di Ubuntu/Debian**
1. Perbarui daftar paket:
```bash
sudo apt update
```
2. Tambahkan repository PHP (opsional, untuk versi terbaru):
```bashsudo add-apt-repository ppa:ondrej/php
sudo apt update
```
3. Instal PHP beserta beberapa ekstensi umum:
```bash
sudo apt install php php-cli php-fpm php-mysql php-xml php-mbstring php-curl php-zip php-gd -y
```
4. Verifikasi instalasi:
```bash
php -v
```

## **2. Installasi PHP di CentOS/RHEL**
1. Perbarui daftar paket:
```bash
sudo yum update
```
2. Tambahkan repository EPEL dan Remi:
```bash
sudo yum install epel-release yum-utils -y
sudo yum install http://rpms.remirepo.net/enterprise/remi-release-7.rpm -y
sudo yum-config-manager --enable remi-php74  # Ganti 74 dengan versi yang diinginkan
```
3. Instal PHP beserta beberapa ekstensi umum:
```bash
sudo yum install php php-cli php-fpm php-mysqlnd php-xml php-mbstring php-curl php-zip php-gd -y
```
4. Verifikasi instalasi:
```bash
php -v
```

## **3. Installasi PHP di Windows**
1. Unduh paket PHP dari situs resmi: [https://windows.php.net/download/](https://windows.php.net/download/)
2. Ekstrak file ZIP ke direktori pilihan Anda, misalnya `C:\php`.
3. Tambahkan direktori PHP ke variabel lingkungan PATH:
   - Buka "System Properties" > "Environment Variables".
   - Di bawah "System variables", pilih "Path" dan klik "Edit".
   - Tambahkan `C:\php` ke daftar.
4. Verifikasi instalasi dengan membuka Command Prompt dan menjalankan:
```bashphp -v
php -v
```

## List Ekstensi PHP
Berikut adalah beberapa ekstensi PHP yang umum digunakan:
- `php-mysql`: Ekstensi untuk koneksi ke database MySQL.
- `php-pgsql`: Ekstensi untuk koneksi ke database PostgreSQL.
- `php-sqlite3`: Ekstensi untuk koneksi ke database SQLite.
- `php-curl`: Ekstensi untuk melakukan permintaan HTTP.
- `php-gd`: Ekstensi untuk manipulasi gambar.
- `php-mbstring`: Ekstensi untuk manipulasi string multibyte.
- `php-xml`: Ekstensi untuk pemrosesan XML.
- `php-zip`: Ekstensi untuk manipulasi file ZIP.
- `php-fpm`: FastCGI Process Manager untuk menjalankan PHP di server web.

>Note: Untuk detail ekstensi lainnya, Anda dapat merujuk ke dokumentasi resmi PHP atau menggunakan perintah `php -m` untuk melihat ekstensi yang terinstal.

## **Kesimpulan**
Instalasi PHP dapat dilakukan dengan mudah di berbagai sistem operasi. Pastikan untuk menginstal ekstensi yang sesuai dengan kebutuhan aplikasi Anda. Setelah instalasi, Anda dapat mulai mengembangkan aplikasi web menggunakan PHP.

# Instalasi Cpanel atau WHM
Panduan ini akan memandu Anda melalui proses instalasi Cpanel atau WHM (WebHost Manager) di server Anda. Pastikan server Anda memenuhi persyaratan sistem sebelum memulai instalasi.

## Persyaratan Sistem
Sebelum menginstal Cpanel atau WHM, pastikan server Anda memenuhi persyaratan berikut:
- Sistem Operasi: CentOS, CloudLinux, atau Red Hat Enterprise Linux (RHEL) versi 7 atau lebih baru.
- Akses Root: Anda harus memiliki akses root ke server Anda.
- Sumber Daya Server: Minimal 1 GB RAM (2 GB atau lebih disarankan) dan 20 GB ruang disk kosong.
- Koneksi Internet: Koneksi internet yang stabil untuk mengunduh paket instalasi.

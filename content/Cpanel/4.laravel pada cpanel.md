# Deploy laravel pada cPanel
laravel adalah salah satu framework PHP yang populer untuk pengembangan aplikasi web. Deploying aplikasi Laravel pada cPanel dapat dilakukan dengan beberapa langkah sederhana. Berikut adalah panduan langkah demi langkah untuk melakukan deploy Laravel pada cPanel.

## Prasyarat
Sebelum memulai proses deploy Laravel pada cPanel, pastikan Anda telah memenuhi prasyarat berikut:
- Akun cPanel aktif dengan akses ke File Manager.
- Aplikasi Laravel yang sudah siap untuk di-deploy.
- Akses ke database MySQL jika aplikasi Laravel Anda memerlukannya.

## Langkah-langkah Deploy Laravel pada cPanel
1. **Persiapkan Aplikasi Laravel**
   - Pastikan aplikasi Laravel Anda sudah siap untuk di-deploy. Jalankan perintah `composer install` untuk menginstal semua dependensi yang diperlukan.

![alt text]()

>Note: Dikarenakan keterbatasan akses SSH pada cPanel, kalian mungkin perlu 
menjalankan perintah ini di lingkungan lokal Anda sebelum mengunggah aplikasi ke cPanel.

   - Buat file `.env` di root direktori aplikasi Laravel Anda dan sesuaikan konfigurasi database serta pengaturan lainnya sesuai kebutuhan.

![alt text]()

2. **Kompres Aplikasi Laravel**
   - Kompres seluruh direktori aplikasi Laravel Anda menjadi file ZIP atau TAR. Ini akan memudahkan proses unggah ke cPanel.

![alt text]()

3. **Unggah Aplikasi ke cPanel**
   - Masuk ke akun cPanel Anda.
   - Buka **File Manager** dan navigasikan ke direktori `public_html` atau direktori tempat Anda ingin meng-host aplikasi Laravel Anda.
   - Unggah file ZIP atau TAR yang telah Anda buat sebelumnya.
   - Setelah unggahan selesai, ekstrak file tersebut di dalam direktori yang diinginkan.

4. **Atur Direktori Publik**
   - Arahkan root dokumen web ke direktori `public` dari aplikasi Laravel Anda. Anda dapat melakukannya dengan mengubah pengaturan di cPanel atau dengan membuat file `.htaccess` di direktori root aplikasi Laravel Anda dengan isi berikut:
   ```
   RewriteEngine On
   RewriteRule ^(.*)$ public/$1 [L]
   ```

5. **Konfigurasi Database**
   - Jika aplikasi Laravel Anda menggunakan database, buat database MySQL baru di cPanel dan catat nama database, username, dan passwordnya.
   - Impor struktur database dan data yang diperlukan menggunakan phpMyAdmin di cPanel.
   - Perbarui file `.env` di aplikasi Laravel Anda dengan informasi database yang baru dibuat.

# Instalasi OpenStack
OpenStack adalah platform cloud computing open-source yang memungkinkan Anda untuk mengelola sumber daya komputasi, penyimpanan, dan jaringan dalam lingkungan cloud. Berikut adalah panduan langkah demi langkah untuk menginstal OpenStack di berbagai sistem operasi.

## **1. Installasi OpenStack di Ubuntu**
1. Perbarui daftar paket:
```bash
sudo apt update
```
2. Instal paket-paket yang diperlukan:
```bash
sudo apt install software-properties-common
```
3. Tambahkan repository OpenStack:
```bash
sudo add-apt-repository cloud-archive:wallaby
sudo apt update && sudo apt dist-upgrade -y
```
4. Instal OpenStack:
```bash
sudo apt install openstack-dashboard
```
5. Konfigurasi OpenStack sesuai kebutuhan Anda.
6. Verifikasi instalasi:
```bash
openstack --version
```

## **2. Installasi OpenStack di CentOS/RHEL**
1. Perbarui daftar paket:
```bash
sudo yum update
```
2. Tambahkan repository OpenStack:
```bash
sudo yum install centos-release-openstack-wallaby
sudo yum update -y
```
3. Instal OpenStack:
```bash
sudo yum install openstack-dashboard
```
4. Konfigurasi OpenStack sesuai kebutuhan Anda.
5. Verifikasi instalasi:
```bash
openstack --version
```

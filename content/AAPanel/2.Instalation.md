# Cara Installasi AAPanel di Linux
Untuk menginstall AAPanel di Linux, Anda dapat mengikuti langkah-langkah berikut:
1. Pilih edisi AAPanel yang ingin Anda install: Free Edition atau Pro Edition.
2. Gunakan perintah berikut sesuai dengan edisi yang Anda pilih:
### PRO Edition
```bash
URL=https://www.aapanel.com/script/install_pro_en.sh && if [ -f /usr/bin/curl ];then curl -ksSO $URL ;else wget --no-check-certificate -O install_pro_en.sh $URL;fi;bash install_pro_en.sh aa372544
```

### FREE Edition
```bash
URL=https://www.aapanel.com/script/install_7.0_en.sh && if [ -f /usr/bin/curl ];then curl -ksSO "$URL" ;else wget --no-check-certificate -O install_7.0_en.sh "$URL";fi;bash install_7.0_en.sh aapanel
```

3. Setelah menjalankan perintah di atas, ikuti petunjuk yang muncul di terminal untuk menyelesaikan proses instalasi.
4. Setelah instalasi selesai, Anda dapat mengakses AAPanel melalui browser web dengan memasukkan alamat IP server Anda diikuti dengan port yang ditentukan selama instalasi (biasanya port 8888).
5. Masuk ke AAPanel menggunakan kredensial yang diberikan selama proses instalasi.
atau Anda juga dapat mengunjungi situs resmi [AAPanel](https://www.aapanel.com/new/download.html) untuk informasi lebih lanjut dan opsi instalasi lainnya.

### Kesimpulan
AAPanel adalah solusi terbaik untuk memanagement server Linux bagi pengguna yang masih awam dengan Linux. Dengan fitur yang lengkap dan mudah digunakan, AAPanel sangat cocok untuk pengguna pemula hingga profesional.

Itulah cara mudah untuk menginstall AAPanel di Linux. Selamat mencoba!
